# Provenance and clean-room statement

HausaQA 0.1.0 was authored as a standalone clean-room project.

- Source code was written specifically for this repository.
- Test and evaluation text is fresh synthetic content created for the project.
- The sample glossary is synthetic.
- No private localization files, client content, proprietary prompts, restricted glossaries, credentials, screenshots, or confidential workflows are included.
- The implementation uses Python standard-library interfaces and general public knowledge about localization file formats and Hausa writing conventions.

Every future contribution must preserve this boundary. Material with uncertain rights or provenance must not be added. Provenance review is part of code review and release preparation.
